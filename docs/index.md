## What Makes This Different

Most awesome lists are **static catalogs** — they list projects that exist. This one tracks projects that are **gaining real momentum**.

Every entry includes:
- **Discovery date** — when we first spotted it trending
- **Momentum signal** — HN points, GitHub stars, community attention at time of discovery
- **Signal rating** — 🔥 Hot / ⚡ Rising / 🌱 Early

### How We Find Projects

Daily research across HackerNews trending, GitHub trending, and developer community signals. Projects are evaluated for:
- Technical substance (not just hype)
- Community adoption velocity
- Problem significance

### Signal Legend

| Emoji | Meaning | Threshold |
|-------|---------|-----------|
| 🔥 | **Hot** | >500 HN points or >10k GitHub stars |
| ⚡ | **Rising** | >100 HN points or >1k GitHub stars |
| 🌱 | **Early** | Promising but nascent |
