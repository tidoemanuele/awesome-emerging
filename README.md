# Awesome Emerging

> A curated list of emerging open source projects that actually matter.

[![Website](https://img.shields.io/badge/Website-awesome--emerging-646cff)](https://tidoemanuele.github.io/awesome-emerging/)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

Unlike static awesome lists that catalog established tools, **Awesome Emerging** tracks projects that are gaining real momentum — discovered through daily research across HackerNews, GitHub trending, and developer communities.

Every entry includes a **discovery date**, **momentum signal** (HN points, GitHub stars), and a **signal rating** (🔥 Hot / ⚡ Rising / 🌱 Early).

## Signal Legend

| Emoji | Meaning | Threshold |
|-------|---------|-----------|
| 🔥 | **Hot** | >500 HN points or >10k GitHub stars |
| ⚡ | **Rising** | >100 HN points or >1k GitHub stars |
| 🌱 | **Early** | Promising but nascent |
